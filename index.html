UBUNTU 4.10 (Warty Warthog)

Ubuntu 4.10 (Warty Warthog), lanzado el 20 de octubre de 2004, es la primera versión de Ubuntu de Canonical, basada en Debian, con planes de sacar una nueva versión cada seis meses y brindar dieciocho meses de soporte posteriormente. Utilizaba el sistema de archivos ext3. El soporte terminó el 30 de abril de 2006. Ubuntu 4.10 se ofreció como descarga gratuita y, a través del servicio ShipIt de Canonical, también se enviaba de forma gratuita a los usuarios en formato de CD.

UBUNTU 5.04 (Hoary Hedgehog)

Ubuntu 5.04 (Hoary Hedgehog), lanzado el 8 de abril de 2005, es la segunda versión de Ubuntu de Canonical. El soporte terminó el 31 de octubre de 2006. Ubuntu 5.04 añadió muchas características y paquetes nuevos, incluyendo la instalación desde dispositivos USB, el Gestor de Actualizaciones, un notificador de actualizaciones, readahead, grepmap, soporte para suspensión, hibernación y modo de espera, escalado dinámico de frecuencia para procesadores, la base de datos de hardware de Ubuntu, instalación Kickstart y autenticación APT. A partir de Ubuntu 5.04, UTF-8 se convirtió en la codificación de caracteres predeterminada.

Ubuntu 5.10 (Breezy Badger)

Ubuntu 5.10 (Breezy Badger), lanzado el 12 de octubre de 2005, es la tercera versión de Ubuntu de Canonical. El soporte terminó el 13 de abril de 2007. Ubuntu 5.10 añadió varias características nuevas, incluyendo un cargador gráfico (Usplash), una herramienta de Agregar/Quitar aplicaciones, un editor de menús (Alacarte), un selector de idioma fácil, soporte para gestión de volúmenes lógicos, soporte completo de impresoras Hewlett-Packard, soporte para instalador OEM, un nuevo logotipo de Ubuntu en la esquina superior izquierda e integración de Launchpad para informes de errores y desarrollo de software.

Ubuntu 6.06 LTS (Dapper Drake)

Ubuntu 6.06 (Dapper Drake), lanzado el 1 de junio de 2006, es la cuarta versión de Ubuntu de Canonical y la primera de soporte a largo plazo (LTS). Originalmente planeada como la versión 6.04, Ubuntu 6.06 fue lanzada con retraso, siendo a veces bromísticamente descrita como su primer lanzamiento "Late To Ship" (LTS). El desarrollo no estaba completo en abril de 2006 y Mark Shuttleworth aprobó posponer la fecha de lanzamiento a junio, convirtiéndola en la versión 6.06 en lugar de 6.04. El soporte terminó el 14 de julio de 2009 para equipos de escritorio y en junio de 2011 para servidores. Ubuntu 6.06 incluyó varias características nuevas, como la fusión del CD Live e Instalación en un solo disco, un instalador gráfico en el CD Live (Ubiquity), Usplash en el apagado y encendido, un gestor de red para cambiar fácilmente entre múltiples conexiones cableadas e inalámbricas, el tema Humanlooks implementado utilizando las pautas de Tango, basado en Clearlooks y presentando colores naranjas en lugar de marrones, y GDebi, un instalador gráfico para archivos de paquetes. Ubuntu 6.06 no incluyó un medio para instalar desde un dispositivo USB, pero permitió por primera vez la instalación directa en dispositivos USB extraíbles.

Ubuntu 6.10 (Edgy Eft)

Ubuntu 6.10 (Edgy Eft), lanzado el 26 de octubre de 2006, es la quinta versión de Ubuntu de Canonical. El soporte terminó el 25 de abril de 2008. Ubuntu 6.10 añadió varias características nuevas, incluyendo un tema Human muy modificado, el iniciador Upstart, informes automatizados de fallos (Apport), la aplicación de toma de notas Tomboy y el gestor de fotos F-Spot. EasyUbuntu, un programa de terceros diseñado para hacer Ubuntu más fácil de usar, se incluyó en Ubuntu 6.10 como un meta-paquete.

Ubuntu 7.04 (Feisty Fawn)

Ubuntu 7.04 (Feisty Fawn), lanzado el 19 de abril de 2007, es la sexta versión de Ubuntu de Canonical. El soporte terminó el 19 de octubre de 2008. Ubuntu 7.04 incluyó varias características nuevas, entre ellas un asistente de migración para ayudar a los antiguos usuarios de Microsoft Windows a hacer la transición a Ubuntu, soporte para Kernel-based Virtual Machine, instalación asistida de códecs y controladores restringidos incluyendo Adobe Flash, Java, soporte para MP3, instalación más sencilla de controladores Nvidia y ATI, efectos de escritorio Compiz, soporte para Wi-Fi Protected Access, la adición de Sudoku y ajedrez, un analizador de uso de disco (baobab), el Centro de Control de GNOME y soporte zeroconf para muchos dispositivos.

Ubuntu 7.10 (Gutsy Gibbon)

Ubuntu 7.10 (Gutsy Gibbon), lanzado el 18 de octubre de 2007, es la séptima versión de Ubuntu de Canonical. El soporte terminó el 18 de abril de 2009. Ubuntu 7.10 incluyó varias características nuevas, entre ellas el marco de seguridad AppArmor, búsqueda rápida en el escritorio, un gestor de complementos para Firefox (Ubufox), una herramienta de configuración gráfica para X.Org, soporte completo de NTFS (lectura/escritura) a través de NTFS-3G y un sistema de impresión renovado con impresión en PDF por defecto. Compiz Fusion se habilitó como predeterminado en Ubuntu 7.10 y se agregó el cambio rápido de usuario.

Ubuntu 8.04 (Hardy Heron)

Ubuntu 8.04 (Hardy Heron), lanzado el 24 de abril de 2008, es la octava versión de Ubuntu de Canonical y la segunda versión de soporte a largo plazo (LTS). El soporte terminó el 12 de mayo de 2011 para equipos de escritorio y el 9 de mayo de 2013 para servidores. Ubuntu 8.04 incluyó varias características nuevas, como la integración de búsqueda en el escritorio Tracker, el grabador de discos Brasero, el cliente BitTorrent Transmission, el cliente VNC Vinagre, sonido del sistema a través de PulseAudio y autenticación y inicio de sesión en Active Directory utilizando Likewise Open.

Ubuntu 8.10 (Intrepid Ibex)

Ubuntu 8.10 (Intrepid Ibex), lanzado el 30 de octubre de 2008, es la novena versión de Ubuntu de Canonical. El soporte terminó el 30 de abril de 2010. Ubuntu 8.10 introdujo varias características nuevas, incluyendo mejoras en la informática móvil y la escalabilidad de escritorio, mayor flexibilidad para la conectividad a Internet, un creador de USB Live de Ubuntu y una cuenta de invitado, que permitía a otros usuarios utilizar una computadora con derechos de usuario muy limitados (por ejemplo, acceso a Internet, uso de software y revisión de correo electrónico). La cuenta de invitado tenía su propio directorio de inicio y nada de lo realizado en ella se almacenaba permanentemente en el disco duro de la computadora.
