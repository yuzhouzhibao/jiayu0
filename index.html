UBUNTU 4.10 (Warty Warthog)

Ubuntu 4.10 (Warty Warthog), lanzado el 20 de octubre de 2004, es la primera versión de Ubuntu de Canonical, basada en Debian, con planes de sacar una nueva versión cada seis meses y brindar dieciocho meses de soporte posteriormente. Utilizaba el sistema de archivos ext3. El soporte terminó el 30 de abril de 2006. Ubuntu 4.10 se ofreció como descarga gratuita y, a través del servicio ShipIt de Canonical, también se enviaba de forma gratuita a los usuarios en formato de CD.

UBUNTU 5.04 (Hoary Hedgehog)

Ubuntu 5.04 (Hoary Hedgehog), lanzado el 8 de abril de 2005, es la segunda versión de Ubuntu de Canonical. El soporte terminó el 31 de octubre de 2006. Ubuntu 5.04 añadió muchas características y paquetes nuevos, incluyendo la instalación desde dispositivos USB, el Gestor de Actualizaciones, un notificador de actualizaciones, readahead, grepmap, soporte para suspensión, hibernación y modo de espera, escalado dinámico de frecuencia para procesadores, la base de datos de hardware de Ubuntu, instalación Kickstart y autenticación APT. A partir de Ubuntu 5.04, UTF-8 se convirtió en la codificación de caracteres predeterminada.

Ubuntu 5.10 (Breezy Badger)

Ubuntu 5.10 (Breezy Badger), lanzado el 12 de octubre de 2005, es la tercera versión de Ubuntu de Canonical. El soporte terminó el 13 de abril de 2007. Ubuntu 5.10 añadió varias características nuevas, incluyendo un cargador gráfico (Usplash), una herramienta de Agregar/Quitar aplicaciones, un editor de menús (Alacarte), un selector de idioma fácil, soporte para gestión de volúmenes lógicos, soporte completo de impresoras Hewlett-Packard, soporte para instalador OEM, un nuevo logotipo de Ubuntu en la esquina superior izquierda e integración de Launchpad para informes de errores y desarrollo de software.

Ubuntu 6.06 LTS (Dapper Drake)

Ubuntu 6.06 (Dapper Drake), lanzado el 1 de junio de 2006, es la cuarta versión de Ubuntu de Canonical y la primera de soporte a largo plazo (LTS). Originalmente planeada como la versión 6.04, Ubuntu 6.06 fue lanzada con retraso, siendo a veces bromísticamente descrita como su primer lanzamiento "Late To Ship" (LTS). El desarrollo no estaba completo en abril de 2006 y Mark Shuttleworth aprobó posponer la fecha de lanzamiento a junio, convirtiéndola en la versión 6.06 en lugar de 6.04. El soporte terminó el 14 de julio de 2009 para equipos de escritorio y en junio de 2011 para servidores.

Ubuntu 6.10 (Edgy Eft)

Ubuntu 6.10 (Edgy Eft), lanzado el 26 de octubre de 2006, es la quinta versión de Ubuntu de Canonical. El soporte terminó el 25 de abril de 2008. Ubuntu 6.10 añadió varias características nuevas, incluyendo un tema Human muy modificado, el iniciador Upstart, informes automatizados de fallos (Apport), la aplicación de toma de notas Tomboy y el gestor de fotos F-Spot. EasyUbuntu, un programa de terceros diseñado para hacer Ubuntu más fácil de usar, se incluyó en Ubuntu 6.10 como un meta-paquete.

Ubuntu 7.04 (Feisty Fawn)

Ubuntu 7.04 (Feisty Fawn), lanzado el 19 de abril de 2007, es la sexta versión de Ubuntu de Canonical. El soporte terminó el 19 de octubre de 2008. Ubuntu 7.04 incluyó varias características nuevas, entre ellas un asistente de migración para ayudar a los antiguos usuarios de Microsoft Windows a hacer la transición a Ubuntu, soporte para Kernel-based Virtual Machine, instalación asistida de códecs y controladores restringidos incluyendo Adobe Flash, Java, soporte para MP3, instalación más sencilla de controladores Nvidia y ATI.

Ubuntu 7.10 (Gutsy Gibbon)

Ubuntu 7.10 (Gutsy Gibbon), lanzado el 18 de octubre de 2007, es la séptima versión de Ubuntu de Canonical. El soporte terminó el 18 de abril de 2009. Ubuntu 7.10 incluyó varias características nuevas, entre ellas el marco de seguridad AppArmor, búsqueda rápida en el escritorio, un gestor de complementos para Firefox (Ubufox), una herramienta de configuración gráfica para X.Org, soporte completo de NTFS (lectura/escritura) a través de NTFS-3G y un sistema de impresión renovado con impresión en PDF por defecto. Compiz Fusion se habilitó como predeterminado en Ubuntu 7.10 y se agregó el cambio rápido de usuario.

Ubuntu 8.04 (Hardy Heron)

Ubuntu 8.04 (Hardy Heron), lanzado el 24 de abril de 2008, es la octava versión de Ubuntu de Canonical y la segunda versión de soporte a largo plazo (LTS). El soporte terminó el 12 de mayo de 2011 para equipos de escritorio y el 9 de mayo de 2013 para servidores. Ubuntu 8.04 incluyó varias características nuevas, como la integración de búsqueda en el escritorio Tracker, el grabador de discos Brasero, el cliente BitTorrent Transmission, el cliente VNC Vinagre, sonido del sistema a través de PulseAudio y autenticación y inicio de sesión en Active Directory utilizando Likewise Open.

Ubuntu 8.10 (Intrepid Ibex)

Ubuntu 8.10 (Intrepid Ibex), lanzado el 30 de octubre de 2008, es la novena versión de Ubuntu de Canonical. El soporte terminó el 30 de abril de 2010. Ubuntu 8.10 introdujo varias características nuevas, incluyendo mejoras en la informática móvil y la escalabilidad de escritorio, mayor flexibilidad para la conectividad a Internet, un creador de USB Live de Ubuntu y una cuenta de invitado, que permitía a otros usuarios utilizar una computadora con derechos de usuario muy limitados (por ejemplo, acceso a Internet, uso de software y revisión de correo electrónico). La cuenta de invitado tenía su propio directorio de inicio y nada de lo realizado en ella se almacenaba permanentemente en el disco duro de la computadora.

Ubuntu 9.04 (Jaunty Jackalope)

Ubuntu 9.04 (Jaunty Jackalope), lanzado el 23 de abril de 2009, es la décima versión de Ubuntu de Canonical. El soporte terminó el 23 de octubre de 2010. Entre las nuevas características se incluyeron un tiempo de arranque más rápido y la integración de servicios y aplicaciones web en la interfaz de escritorio. Debido a esto, se le puso el nombre del mitológico jackalope. Fue la primera versión nombrada después de un animal mítico, siendo la segunda el Utopic Unicorn. Tenía una nueva pantalla de inicio, una nueva pantalla de inicio de sesión y soporte tanto para tabletas Wacom (hotplugging) como para netbooks.

Ubuntu 9.10 (Karmic Koala)

Ubuntu 9.10 (Karmic Koala), lanzado el 29 de octubre de 2009, es la undécima versión de Ubuntu de Canonical. El soporte terminó en abril de 2011. En la instalación de escritorio de Ubuntu 9.10, se reemplazó Pidgin con Empathy Instant Messenger como cliente de mensajería instantánea predeterminado. El sistema de archivos predeterminado es ext4 y el cliente de Ubuntu One, que se conecta con el nuevo sistema de almacenamiento en línea de Canonical, se instala por defecto. Introdujo Grub 2 beta como cargador de arranque predeterminado.

Ubuntu 10.04 (Lucid Lynx)

Ubuntu 10.04 (Lucid Lynx) fue anunciado por primera vez por Shuttleworth el 19 de septiembre de 2009 en el Atlanta Linux Fest antes de su lanzamiento el 29 de abril de 2010. Es la 12ª versión de Ubuntu de Canonical y la tercera versión de soporte a largo plazo (LTS). Canonical brindó soporte para la versión de escritorio de Ubuntu 10.04 hasta el 9 de mayo de 2013 y para la versión de servidor hasta el 30 de abril de 2015. Esta versión incluyó mejoras en el soporte para controladores gráficos propietarios de Nvidia al cambiar al controlador gráfico de código abierto Nvidia, Nouveau, por defecto. Se introdujo Plymouth, permitiendo animaciones de arranque.

Ubuntu 10.10 (Maverick Meerkat)

El nombre de Ubuntu 10.10 (Maverick Meerkat) fue anunciado por Shuttleworth el 2 de abril de 2010, junto con los objetivos del lanzamiento de mejorar la experiencia en netbooks y enfocarse en la computación en la nube híbrida para servidores. Ubuntu 10.10 fue lanzado el 10 de octubre de 2010 (10.10.10) alrededor de las 10:10 UTC. Este lanzamiento se apartó del horario tradicional de lanzamientos a finales de octubre para alcanzar "la perfección del 10" y fue una referencia lúdica a "The Hitchhiker's Guide to the Galaxy".

Ubuntu 11.04 (Natty Narwhal)

El nombre de Ubuntu 11.04 (Natty Narwhal) fue anunciado el 17 de agosto de 2010 por Mark Shuttleworth. Fue lanzado el 28 de abril de 2011 y es la 14ª versión de Ubuntu de Canonical. El soporte terminó el 28 de octubre de 2012. Una de las características más destacadas de Ubuntu 11.04 fue el cambio a la interfaz de usuario Unity en lugar de GNOME 2 como predeterminada. Este cambio fue controversial, ya que algunos desarrolladores de GNOME temían que dividiera la comunidad y marginara GNOME Shell. Ubuntu 11.04 empleó Banshee como reproductor de música predeterminado, reemplazando a Rhythmbox.

Ubuntu 11.10 (Oneiric Ocelot)

El nombre de Ubuntu 11.10 (Oneiric Ocelot) fue anunciado el 7 de marzo de 2011 por Mark Shuttleworth, explicando que "Oneiric" significa "onírico" o "de ensueño". Fue lanzado el 13 de octubre de 2011 y es la 15ª versión de Ubuntu de Canonical. El soporte terminó el 9 de mayo de 2013. En abril de 2011, Shuttleworth anunció que Ubuntu 11.10 no incluiría el escritorio clásico de GNOME como alternativa a Unity. En cambio, 11.10 incluyó una versión 2D de Unity como alternativa para computadoras que carecían de los recursos de hardware para la versión 3D basada en Compiz.
