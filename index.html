UBUNTU 5.04 (Hoary Hedgehog)

Ubuntu 5.04 (Hoary Hedgehog), lanzado el 8 de abril de 2005, es la segunda versión de Ubuntu de Canonical. El soporte terminó el 31 de octubre de 2006. Ubuntu 5.04 añadió muchas características y paquetes nuevos, incluyendo la instalación desde dispositivos USB, el Gestor de Actualizaciones, un notificador de actualizaciones, readahead, grepmap, soporte para suspensión, hibernación y modo de espera, escalado dinámico de frecuencia para procesadores, la base de datos de hardware de Ubuntu, instalación Kickstart y autenticación APT. A partir de Ubuntu 5.04, UTF-8 se convirtió en la codificación de caracteres predeterminada.

Ubuntu 5.10 (Breezy Badger)

Ubuntu 5.10 (Breezy Badger), lanzado el 12 de octubre de 2005, es la tercera versión de Ubuntu de Canonical. El soporte terminó el 13 de abril de 2007. Ubuntu 5.10 añadió varias características nuevas, incluyendo un cargador gráfico (Usplash), una herramienta de Agregar/Quitar aplicaciones, un editor de menús (Alacarte), un selector de idioma fácil, soporte para gestión de volúmenes lógicos, soporte completo de impresoras Hewlett-Packard, soporte para instalador OEM, un nuevo logotipo de Ubuntu en la esquina superior izquierda e integración de Launchpad para informes de errores y desarrollo de software.

Ubuntu 6.06 LTS (Dapper Drake)

Ubuntu 6.06 (Dapper Drake), lanzado el 1 de junio de 2006, es la cuarta versión de Ubuntu de Canonical y la primera de soporte a largo plazo (LTS). Originalmente planeada como la versión 6.04, Ubuntu 6.06 fue lanzada con retraso, siendo a veces bromísticamente descrita como su primer lanzamiento "Late To Ship" (LTS). El desarrollo no estaba completo en abril de 2006 y Mark Shuttleworth aprobó posponer la fecha de lanzamiento a junio, convirtiéndola en la versión 6.06 en lugar de 6.04. El soporte terminó el 14 de julio de 2009 para equipos de escritorio y en junio de 2011 para servidores. Ubuntu 6.06 incluyó varias características nuevas, como la fusión del CD Live e Instalación en un solo disco, un instalador gráfico en el CD Live (Ubiquity), Usplash en el apagado y encendido, un gestor de red para cambiar fácilmente entre múltiples conexiones cableadas e inalámbricas, el tema Humanlooks implementado utilizando las pautas de Tango, basado en Clearlooks y presentando colores naranjas en lugar de marrones, y GDebi, un instalador gráfico para archivos de paquetes. Ubuntu 6.06 no incluyó un medio para instalar desde un dispositivo USB, pero permitió por primera vez la instalación directa en dispositivos USB extraíbles.
