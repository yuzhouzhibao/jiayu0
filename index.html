UBUNTU 4.10 (Warty Warthog)

Ubuntu 4.10 (Warty Warthog), lanzado el 20 de octubre de 2004, es la primera versión de Ubuntu de Canonical, basada en Debian, con planes de sacar una nueva versión cada seis meses y brindar dieciocho meses de soporte posteriormente. Utilizaba el sistema de archivos ext3. El soporte terminó el 30 de abril de 2006. Ubuntu 4.10 se ofreció como descarga gratuita y, a través del servicio ShipIt de Canonical, también se enviaba de forma gratuita a los usuarios en formato de CD.

UBUNTU 5.04 (Hoary Hedgehog)

Ubuntu 5.04 (Hoary Hedgehog), lanzado el 8 de abril de 2005, es la segunda versión de Ubuntu de Canonical. El soporte terminó el 31 de octubre de 2006. Ubuntu 5.04 añadió muchas características y paquetes nuevos, incluyendo la instalación desde dispositivos USB, el Gestor de Actualizaciones, un notificador de actualizaciones, readahead, grepmap, soporte para suspensión, hibernación y modo de espera, escalado dinámico de frecuencia para procesadores, la base de datos de hardware de Ubuntu, instalación Kickstart y autenticación APT. A partir de Ubuntu 5.04, UTF-8 se convirtió en la codificación de caracteres predeterminada.

Ubuntu 5.10 (Breezy Badger)

Ubuntu 5.10 (Breezy Badger), lanzado el 12 de octubre de 2005, es la tercera versión de Ubuntu de Canonical. El soporte terminó el 13 de abril de 2007. Ubuntu 5.10 añadió varias características nuevas, incluyendo un cargador gráfico (Usplash), una herramienta de Agregar/Quitar aplicaciones, un editor de menús (Alacarte), un selector de idioma fácil, soporte para gestión de volúmenes lógicos, soporte completo de impresoras Hewlett-Packard, soporte para instalador OEM, un nuevo logotipo de Ubuntu en la esquina superior izquierda e integración de Launchpad para informes de errores y desarrollo de software.

Ubuntu 6.06 LTS (Dapper Drake)

Ubuntu 6.06 (Dapper Drake), lanzado el 1 de junio de 2006, es la cuarta versión de Ubuntu de Canonical y la primera de soporte a largo plazo (LTS). Originalmente planeada como la versión 6.04, Ubuntu 6.06 fue lanzada con retraso, siendo a veces bromísticamente descrita como su primer lanzamiento "Late To Ship" (LTS). El desarrollo no estaba completo en abril de 2006 y Mark Shuttleworth aprobó posponer la fecha de lanzamiento a junio, convirtiéndola en la versión 6.06 en lugar de 6.04. El soporte terminó el 14 de julio de 2009 para equipos de escritorio y en junio de 2011 para servidores.

Ubuntu 6.10 (Edgy Eft)

Ubuntu 6.10 (Edgy Eft), lanzado el 26 de octubre de 2006, es la quinta versión de Ubuntu de Canonical. El soporte terminó el 25 de abril de 2008. Ubuntu 6.10 añadió varias características nuevas, incluyendo un tema Human muy modificado, el iniciador Upstart, informes automatizados de fallos (Apport), la aplicación de toma de notas Tomboy y el gestor de fotos F-Spot. EasyUbuntu, un programa de terceros diseñado para hacer Ubuntu más fácil de usar, se incluyó en Ubuntu 6.10 como un meta-paquete.

Ubuntu 7.04 (Feisty Fawn)

Ubuntu 7.04 (Feisty Fawn), lanzado el 19 de abril de 2007, es la sexta versión de Ubuntu de Canonical. El soporte terminó el 19 de octubre de 2008. Ubuntu 7.04 incluyó varias características nuevas, entre ellas un asistente de migración para ayudar a los antiguos usuarios de Microsoft Windows a hacer la transición a Ubuntu, soporte para Kernel-based Virtual Machine, instalación asistida de códecs y controladores restringidos incluyendo Adobe Flash, Java, soporte para MP3, instalación más sencilla de controladores Nvidia y ATI.

Ubuntu 7.10 (Gutsy Gibbon)

Ubuntu 7.10 (Gutsy Gibbon), lanzado el 18 de octubre de 2007, es la séptima versión de Ubuntu de Canonical. El soporte terminó el 18 de abril de 2009. Ubuntu 7.10 incluyó varias características nuevas, entre ellas el marco de seguridad AppArmor, búsqueda rápida en el escritorio, un gestor de complementos para Firefox (Ubufox), una herramienta de configuración gráfica para X.Org, soporte completo de NTFS (lectura/escritura) a través de NTFS-3G y un sistema de impresión renovado con impresión en PDF por defecto. Compiz Fusion se habilitó como predeterminado en Ubuntu 7.10 y se agregó el cambio rápido de usuario.

Ubuntu 8.04 (Hardy Heron)

Ubuntu 8.04 (Hardy Heron), lanzado el 24 de abril de 2008, es la octava versión de Ubuntu de Canonical y la segunda versión de soporte a largo plazo (LTS). El soporte terminó el 12 de mayo de 2011 para equipos de escritorio y el 9 de mayo de 2013 para servidores. Ubuntu 8.04 incluyó varias características nuevas, como la integración de búsqueda en el escritorio Tracker, el grabador de discos Brasero, el cliente BitTorrent Transmission, el cliente VNC Vinagre, sonido del sistema a través de PulseAudio y autenticación y inicio de sesión en Active Directory utilizando Likewise Open.

Ubuntu 8.10 (Intrepid Ibex)

Ubuntu 8.10 (Intrepid Ibex), lanzado el 30 de octubre de 2008, es la novena versión de Ubuntu de Canonical. El soporte terminó el 30 de abril de 2010. Ubuntu 8.10 introdujo varias características nuevas, incluyendo mejoras en la informática móvil y la escalabilidad de escritorio, mayor flexibilidad para la conectividad a Internet, un creador de USB Live de Ubuntu y una cuenta de invitado, que permitía a otros usuarios utilizar una computadora con derechos de usuario muy limitados (por ejemplo, acceso a Internet, uso de software y revisión de correo electrónico). La cuenta de invitado tenía su propio directorio de inicio y nada de lo realizado en ella se almacenaba permanentemente en el disco duro de la computadora.

Ubuntu 9.04 (Jaunty Jackalope)

Ubuntu 9.04 (Jaunty Jackalope), lanzado el 23 de abril de 2009, es la décima versión de Ubuntu de Canonical. El soporte terminó el 23 de octubre de 2010. Entre las nuevas características se incluyeron un tiempo de arranque más rápido y la integración de servicios y aplicaciones web en la interfaz de escritorio. Debido a esto, se le puso el nombre del mitológico jackalope. Fue la primera versión nombrada después de un animal mítico, siendo la segunda el Utopic Unicorn. Tenía una nueva pantalla de inicio, una nueva pantalla de inicio de sesión y soporte tanto para tabletas Wacom (hotplugging) como para netbooks.

Ubuntu 9.10 (Karmic Koala)

Ubuntu 9.10 (Karmic Koala), lanzado el 29 de octubre de 2009, es la undécima versión de Ubuntu de Canonical. El soporte terminó en abril de 2011. En la instalación de escritorio de Ubuntu 9.10, se reemplazó Pidgin con Empathy Instant Messenger como cliente de mensajería instantánea predeterminado. El sistema de archivos predeterminado es ext4 y el cliente de Ubuntu One, que se conecta con el nuevo sistema de almacenamiento en línea de Canonical, se instala por defecto. Introdujo Grub 2 beta como cargador de arranque predeterminado.

Ubuntu 10.04 (Lucid Lynx)

Ubuntu 10.04 (Lucid Lynx) fue anunciado por primera vez por Shuttleworth el 19 de septiembre de 2009 en el Atlanta Linux Fest antes de su lanzamiento el 29 de abril de 2010. Es la 12ª versión de Ubuntu de Canonical y la tercera versión de soporte a largo plazo (LTS). Canonical brindó soporte para la versión de escritorio de Ubuntu 10.04 hasta el 9 de mayo de 2013 y para la versión de servidor hasta el 30 de abril de 2015. Esta versión incluyó mejoras en el soporte para controladores gráficos propietarios de Nvidia al cambiar al controlador gráfico de código abierto Nvidia, Nouveau, por defecto. Se introdujo Plymouth, permitiendo animaciones de arranque.

Ubuntu 10.10 (Maverick Meerkat)

El nombre de Ubuntu 10.10 (Maverick Meerkat) fue anunciado por Shuttleworth el 2 de abril de 2010, junto con los objetivos del lanzamiento de mejorar la experiencia en netbooks y enfocarse en la computación en la nube híbrida para servidores. Ubuntu 10.10 fue lanzado el 10 de octubre de 2010 (10.10.10) alrededor de las 10:10 UTC. Este lanzamiento se apartó del horario tradicional de lanzamientos a finales de octubre para alcanzar "la perfección del 10" y fue una referencia lúdica a "The Hitchhiker's Guide to the Galaxy".

Ubuntu 11.04 (Natty Narwhal)

El nombre de Ubuntu 11.04 (Natty Narwhal) fue anunciado el 17 de agosto de 2010 por Mark Shuttleworth. Fue lanzado el 28 de abril de 2011 y es la 14ª versión de Ubuntu de Canonical. El soporte terminó el 28 de octubre de 2012. Una de las características más destacadas de Ubuntu 11.04 fue el cambio a la interfaz de usuario Unity en lugar de GNOME 2 como predeterminada. Este cambio fue controversial, ya que algunos desarrolladores de GNOME temían que dividiera la comunidad y marginara GNOME Shell. Ubuntu 11.04 empleó Banshee como reproductor de música predeterminado, reemplazando a Rhythmbox.

Ubuntu 11.10 (Oneiric Ocelot)

El nombre de Ubuntu 11.10 (Oneiric Ocelot) fue anunciado el 7 de marzo de 2011 por Mark Shuttleworth, explicando que "Oneiric" significa "onírico" o "de ensueño". Fue lanzado el 13 de octubre de 2011 y es la 15ª versión de Ubuntu de Canonical. El soporte terminó el 9 de mayo de 2013. En abril de 2011, Shuttleworth anunció que Ubuntu 11.10 no incluiría el escritorio clásico de GNOME como alternativa a Unity. En cambio, 11.10 incluyó una versión 2D de Unity como alternativa para computadoras que carecían de los recursos de hardware para la versión 3D basada en Compiz.

Ubuntu 12.04 LTS (Precise Pangolin)

Ubuntu 12.04 LTS (Precise Pangolin) es la 16ª versión de Ubuntu de Canonical y su cuarta versión de soporte a largo plazo (LTS), lanzada el 26 de abril de 2012. Recibe su nombre del pangolín, un tipo de hormiguero. A diferencia de las versiones LTS anteriores que contaron con tres años de soporte para la versión de escritorio y cinco años para la versión de servidor, esta versión tuvo un soporte de cinco años para ambas versiones, finalizando el soporte el 28 de abril de 2017. Canonical siguió ofreciendo mantenimiento extendido de seguridad a los clientes Advantage durante dos años adicionales.

Ubuntu 13.04 (Raring Ringtail)

El 17 de octubre de 2012, Shuttleworth anunció que Ubuntu 13.04 (Raring Ringtail) se centraría en "métricas móviles, como la duración de la batería, el número de procesos en ejecución, el uso de memoria, y en pulir los detalles ásperos que encontramos cuando hacemos eso". Fue lanzado el 25 de abril de 2013 y el soporte terminó el 27 de enero de 2014. Esta versión eliminó el instalador Wubi para Windows debido a su incompatibilidad con Windows 8 y a la falta general de soporte y desarrollo. Incluyó Unity 7, que tuvo muchas mejoras de rendimiento, y permitió la búsqueda de fotos y publicaciones en redes sociales desde el Dash.

Ubuntu 13.10 (Saucy Salamander)

Ubuntu 13.10 (Saucy Salamander), lanzado el 17 de octubre de 2013, es la 19ª versión de Ubuntu de Canonical. El soporte terminó el 17 de julio de 2014. Se consideró cambiar el navegador predeterminado de Mozilla Firefox a Chromium, pero problemas con las actualizaciones del paquete de Chromium de Ubuntu llevaron a los desarrolladores a mantener Firefox para esta versión. De manera similar, se tenía la intención de reemplazar el antiguo sistema de ventanas X Window System (X11) con el servidor de visualización Mir, permitiendo que los programas X11 operaran a través de la capa de compatibilidad XMir. Sin embargo, tras enfrentar "dificultades técnicas pendientes" con XMir en configuraciones de múltiples monitores, Canonical decidió posponer el uso predeterminado de Mir en Ubuntu. Mir fue lanzado como el servidor de visualización predeterminado para Ubuntu Touch 13.10.

Ubuntu 14.04 LTS (Trusty Tahr)

Mark Shuttleworth anunció el 31 de octubre de 2011 que para Ubuntu 14.04, la plataforma daría soporte a teléfonos inteligentes, tabletas, televisores y pantallas inteligentes. El 18 de octubre de 2013, Shuttleworth anunció que Ubuntu 14.04 (Trusty Tahr) se centraría en "rendimiento, refinamiento, mantenibilidad y deuda técnica" y alentó a los desarrolladores a tomar "decisiones conservadoras". Esta versión, la 20ª de Ubuntu, se lanzó el 17 de abril de 2014 y el soporte finalizó el 25 de abril de 2019. Luego, el mantenimiento extendido de seguridad estuvo disponible para clientes de Ubuntu Advantage durante dos años adicionales. Sin embargo, en septiembre de 2021, Canonical anunció que extendería el soporte LTS para las versiones 14.04 y 16.04 a un total de 10 años, extendiendo el soporte ESM para 14.04 hasta abril de 2024.

Ubuntu 14.10 (Utopic Unicorn)

El 23 de abril de 2014, Shuttleworth anunció que Ubuntu 14.10 llevaría el nombre de Utopic Unicorn. Esta versión es la 21ª publicación, oficialmente caracterizada como una versión que abordaba "correcciones de errores y mejoras incrementales de calidad". Ubuntu 15.04 (Vivid Vervet)

El 20 de octubre de 2014, Shuttleworth anunció que Ubuntu 15.04 llevaría el nombre de Vivid Vervet. Fue lanzado el 23 de abril de 2015 y es la 22ª versión de Ubuntu. Utilizó systemd en lugar de Upstart de manera predeterminada. Jesse Smith de DistroWatch elogió la estabilidad de esta versión, especialmente en medio del cambio a systemd. Esta versión también presentaba menús integrados localmente de manera predeterminada, reemplazando los anteriores menús globales por defecto. Incluyó mejoras modestas en el rendimiento gráfico de Intel Haswell y mejoras significativas para las tarjetas gráficas AMD Radeon que utilizan los controladores de código abierto Radeon R600 y RadeonSI Gallium3D.

Ubuntu 15.10 (Wily Werewolf)

Shuttleworth anunció el 4 de mayo de 2015 que Ubuntu 15.10 se llamaría Wily Werewolf. Inicialmente, expresó la esperanza de que el lanzamiento incluyera el servidor de visualización Mir, pero finalmente se lanzó el 22 de octubre de 2015 sin Mir. Es la 23ª versión de Ubuntu y eliminó las barras de desplazamiento de borde de ventana en favor de las barras de desplazamiento upstream de GNOME, un cambio destinado a ahorrar tiempo a los desarrolladores en la creación de parches y actualizaciones. Steven J. Vaughan-Nichols de ZDNet elogió el lanzamiento por su integración de servicios en la nube, como la nueva herramienta de implementación y gestión de nubes de Ubuntu "OpenStack Autopilot", así como sus herramientas de servidor, especialmente el hipervisor de contenedores de máquinas de Ubuntu, LXD, incluido por defecto en la versión 15.10.

Ubuntu 16.04 LTS (Xenial Xerus)

Shuttleworth anunció el 21 de octubre de 2015 que Ubuntu 16.04 LTS se llamaría Xenial Xerus e incluiría una opción para Unity 8. Fue lanzado el 21 de abril de 2016. En septiembre de 2021, Canonical anunció que extendería el soporte LTS para las versiones 14.04 y 16.04 a un total de 10 años, extendiendo la fecha de soporte ESM para 16.04 hasta abril de 2026. Esta versión añade soporte para los sistemas de archivos Ceph y ZFS, el hipervisor LXD para OpenStack y paquetes Snap. Utiliza systemd en lugar de Upstart como su sistema init. En esta versión, los resultados de búsqueda en línea del Dash están deshabilitados por defecto en Unity 7, no admite el controlador AMD Catalyst (fglrx) para tarjetas gráficas AMD/ATI y en su lugar recomienda alternativas como Radeon y AMDGPU.

Ubuntu 16.10 (Yakkety Yak)

"Yakkety Yak" es una versión de Ubuntu. Mark Shuttleworth anunció el 21 de abril de 2016 que Ubuntu 16.10 llevaría el nombre de Yakkety Yak. Fue lanzado el 13 de octubre de 2016. Esta versión incluye una versión más rápida de Ubuntu Software, un mejor soporte para instalar aplicaciones solo de línea de comandos, soporte para la instalación de fuentes y códecs multimedia, aplicaciones de pago, entradas de registro de cambios para Archivos de Paquetes Personales (PPAs) en el Gestor de Actualizaciones, gestión de la sesión de usuario por systemd y el kernel Linux 4.8.

Ubuntu 17.04 (Zesty Zapus)

El 17 de octubre de 2016, Mark Shuttleworth anunció que el nombre en clave de Ubuntu 17.04, lanzado el 13 de abril de 2017, sería Zesty Zapus. Esta versión dejó de ofrecer soporte para la arquitectura de 32 bits PowerPC, siguiendo el mismo movimiento del proyecto Debian principal. Otros cambios incluyen el resolutor DNS predeterminado que ahora es systemd-resolved, el kernel Linux 4.10 y el soporte incluido para impresoras. Los revisores notaron que esta probablemente sería la última versión de Ubuntu que se enviaría con Unity 7 de manera predeterminada antes del cambio de Ubuntu a GNOME, coincidiendo con el final del alfabeto en el esquema de nombres en clave de Ubuntu.

Ubuntu 17.10 (Artful Aardvark)

Artful Aardvark, la 27ª versión de Ubuntu, fue anunciada a través de Launchpad el 21 de abril de 2017 en lugar de en el blog de Shuttleworth, como había sido el caso en el pasado. Se lanzó el 19 de octubre de 2017. Los críticos elogiaron la transición fluida a GNOME y la importancia de los cambios en esta versión. Esta es la primera versión de Ubuntu que utiliza la interfaz GNOME Shell y la primera en reemplazar X11 con el servidor de visualización Wayland. En mayo de 2017, Ken VanDine, un ingeniero de software de Canonical en el equipo de escritorio de Ubuntu encargado del cambio a GNOME, confirmó que la intención es enviar la versión más actual de GNOME, con muy pocos cambios en comparación con una instalación estándar. Esta versión también eliminó las imágenes de escritorio de 32 bits; sin embargo, las versiones de 32 bits de otras imágenes siguen existiendo.

Ubuntu 18.04 LTS (Bionic Beaver)

Ubuntu 18.04 LTS Bionic Beaver, la séptima versión LTS, es una versión de soporte a largo plazo que fue anunciada el 24 de octubre de 2017 en el blog de Shuttleworth y lanzada el 26 de abril de 2018. Ubuntu 18.04 LTS tuvo soporte LTS normal durante cinco años hasta mayo de 2023 y cuenta con soporte ESM pago disponible de Canonical durante cinco años adicionales hasta abril de 2028. Entre las nuevas características se incluyen emoticonos en color, una nueva aplicación de tareas preinstalada en la instalación por defecto, la opción de "Instalación Mínima" en el instalador del sistema, que instala solo un navegador web y herramientas del sistema, y un instalador de sistema de línea de comandos con marca. Esta versión utilizó el kernel Linux 4.15, que incorporaba un controlador de CPU para la interfaz cgroup v2, soporte para encriptación segura de memoria AMD y mejoras en la Gestión de Energía del Enlace SATA.

Ubuntu 18.10 (Cosmic Cuttlefish)

Shuttleworth anunció Ubuntu 18.10 Cosmic Cuttlefish el 8 de mayo de 2018. Fue lanzado el 18 de octubre de 2018. Las velocidades de instalación son más rápidas debido al uso de un algoritmo de compresión sin pérdidas conocido como Zstandard. También se mejoraron las velocidades de inicio de las aplicaciones Snap preinstaladas. Ubuntu 18.10 incluye un nuevo tema, Yaru, como tema predeterminado, junto con su tema de iconos correspondiente, Suru.

Ubuntu 19.04 (Disco Dingo)

Ubuntu 19.04, apodado Disco Dingo, fue lanzado el 18 de abril de 2019. Incorpora el kernel Linux 5.0, que agrega soporte para la tecnología AMD FreeSync en pantallas de cristal líquido, pantallas táctiles Raspberry Pi, encriptación Adiantum, archivos de intercambio Btrfs, así como numerosas mejoras en USB 3.2 y Type-C, y otros nuevos dispositivos de hardware. Utiliza GNOME 3.32, que incluye un nuevo conjunto de iconos, control de intensidad de luz nocturna, permisos avanzados de aplicaciones, favoritos de archivos, una nueva barra de encabezado, así como modos de 'buscar' y 'solo lectura' en el emulador de terminal predeterminado. La versión 19 de los controladores gráficos de código abierto Mesa está disponible nativamente en esta versión de Ubuntu.

Ubuntu 19.10 (Eoan Ermine)

Ubuntu 19.10, apodado "Eoan Ermine" (/iːˈoʊən/), fue lanzado el 17 de octubre de 2019. Utiliza el kernel Linux 5.3, que entre otras cosas, introduce compatibilidad para placas base de CPU Ryzen de tercera generación y dispositivos inalámbricos asociados de Intel, así como las GPU AMD Navi de 7 nm. El soporte experimental para el sistema de archivos ZFS ahora está disponible desde el instalador. Se realizaron mejoras específicas para NVIDIA. Los controladores gráficos propietarios de Nvidia ahora se incluyen en el instalador en lugar de los controladores de código abierto Nouveau. Se añadió soporte para la plataforma Raspberry Pi 4.

Ubuntu 20.04 LTS (Focal Fossa)

Ubuntu 20.04 LTS, apodado Focal Fossa, es una versión de soporte a largo plazo que se lanzó el 23 de abril de 2020. Como versión LTS, proporcionará actualizaciones de mantenimiento durante 5 años, hasta abril de 2025. Esta versión se basa en el kernel Linux 5.4 LTS, que agrega soporte para nuevo hardware, incluidas las CPUs Intel Comet Lake y Tiger Lake, los SoCs Snapdragon 835 y 855 de Qualcomm, y las GPUs AMD Navi 12 y 14. También añade soporte para actualizaciones del kernel sin reiniciar, el sistema de archivos exFAT, la VPN WireGuard de código abierto y un módulo de seguridad llamado Lockdown, desactivado por defecto, que tiene como objetivo evitar que las cuentas privilegiadas de root interactúen con el kernel subyacente al restringir la funcionalidad del kernel, prohibir la ejecución de código arbitrario y hacer cumplir las firmas de los módulos del kernel.

Ubuntu 20.10 (Groovy Gorilla)

Ubuntu 20.10, apodado Groovy Gorilla, se lanzó el 22 de octubre de 2020. Esta versión se basa en GNOME 3.38 y en el kernel Linux 5.8, que incluye soporte para USB4, CPUs AMD Zen 3, procesadores Intel Ice Lake y Tiger Lake, y soporte inicial para el arranque de procesadores Power10. Una cadena de herramientas actualizada incluye glibc 2.32, OpenJDK 11, rustc 1.41, GCC 10, LLVM 11, Python 3.8.6, ruby 2.7.0, php 7.4.9, perl 5.30 y golang 1.13. Además, nftables es ahora el backend de firewall predeterminado, reemplazando a iptables. Ubuntu 20.10 es la primera versión que incluye imágenes de escritorio para el Raspberry Pi 4 (modelos de 4GB y 8GB) y el Compute Module 4. Los modelos de Pi más antiguos con menos memoria no tienen soporte oficial.

Ubuntu 21.04 (Hirsute Hippo)

Ubuntu 21.04, apodado Hirsute Hippo, se lanzó el 22 de abril de 2021. Utiliza el kernel Linux 5.11, que introduce la autenticación de tarjetas inteligentes, el soporte para las Extensiones de Guardia de Software de Intel y mejoras en el soporte para CPUs y GPUs de AMD. Wayland se utiliza ahora de manera predeterminada en hardware sin procesadores gráficos Nvidia. También se agregó soporte para arrastrar y soltar desde el administrador de archivos al escritorio.

Ubuntu 21.10 (Impish Indri)

Ubuntu 21.10, apodado Impish Indri, se lanzó el 14 de octubre de 2021. Utiliza el kernel Linux 5.13, que introduce un soporte rudimentario para los chips Apple M1, soporte FreeSync HDMI para GPUs AMD, un nuevo módulo de seguridad llamado 'Landlock' y compatibilidad con varios nuevos dispositivos de hardware. Esta versión hace la transición de GNOME 3.38 a GNOME 40, que introduce un conmutador horizontal de espacios de trabajo y un diseño mejorado de la Visión general de actividades. El Dock de Ubuntu sigue estando verticalmente ubicado a la izquierda de la pantalla y ahora presenta separadores entre las aplicaciones fijadas y en ejecución, un icono persistente de la papelera de reciclaje y accesos directos a unidades USB.

Ubuntu 22.04 LTS (Jammy Jellyfish)

Ubuntu 22.04, con el nombre en código de Jammy Jellyfish, se lanzó el 21 de abril de 2022 y es una versión de soporte a largo plazo, con soporte durante cinco años, hasta abril de 2027. Ubuntu 22.04 LTS Desktop utiliza el kernel Linux 5.17 para hardware más nuevo y un kernel HWE (habilitación de hardware) en constante actualización basado en la versión 5.15 para otro hardware; Ubuntu 22.04 LTS Server usa la versión 5.15, mientras que Ubuntu Cloud y Ubuntu para IoT utilizan un kernel optimizado basado en la versión 5.15. Se actualiza Python a 3.10 y Ruby a 3.0.

Ubuntu 22.10 (Kinetic Kudu)

Ubuntu 22.10, con el nombre en clave Kinetic Kudu, fue lanzado el 20 de octubre de 2022. Utiliza el kernel de Linux 5.19, que mejora la eficiencia energética en computadoras basadas en Intel y admite la descompresión multihilo. También actualiza a GNOME 43, agrega soporte para MicroPython en microcontroladores como el Raspberry Pi Pico W y añade soporte para procesadores RISC-V. rshell, thonny y mpremote se agregaron a los repositorios de Ubuntu.

Ubuntu 23.04 (Lunar Lobster)

Ubuntu 23.04, denominado Lunar Lobster, es una versión interina lanzada el 20 de abril de 2023 y tiene soporte durante nueve meses hasta el 20 de enero de 2024. Esta versión incorpora GNOME 44, el kernel de Linux 6.2, controladores gráficos Mesa 23.0, un nuevo instalador basado en Flutter, un menú de Configuración Rápida mejorado, un nuevo menú de Mouse y Touchpad en Configuraciones, tiempos de inicio mejorados para paquetes Snap y un soporte mejorado para paquetes Snap que permite descargar aplicaciones abiertas en segundo plano e instalarlas cuando se cierra la aplicación. El lanzamiento también proporciona soporte para Microsoft Azure Active Directory (también conocido como Entra ID), que permite a los usuarios con planes de empresa de Microsoft 365 autenticar los escritorios de Ubuntu utilizando credenciales comunes.

Ubuntu 23.10 (Mantic Minotaur)

buntu 23.10 Mantic Minotaur es una versión interina, lanzada originalmente el 12 de octubre de 2023 y con soporte durante nueve meses hasta julio de 2024. Este lanzamiento incorpora GNOME 45, un nuevo Centro de Aplicaciones construido en Flutter que reemplaza a Ubuntu Software, cifrado de disco TPM y un actualizador de firmware separado. Aproximadamente 6 horas después del lanzamiento, el enlace de descarga de Ubuntu 23.10 fue eliminado debido a discursos de odio en una traducción externa en ucraniano del instalador. Las descargas fueron restauradas 4 días después.

Ubuntu 24.04 LTS (Noble Numbat)

Ubuntu 24.04 LTS, llamado Noble Numbat, está programado para su lanzamiento el 25 de abril de 2024.
